Physical Based Shader on WebGL

A physical based rendering using Cook-Torrance mircofacet specular BRDF. User can tweak around roughness and fresnel. In this project, I also use the Image-Based Lighting. Live Demo github

NPR Tile Based Deferred Shading on WebGL

We make deferred shading happen on WebGL and improve the performance using tile-based method. Also, we combined with Chinese Panting NPR effect. This is the very first WebGL app using draw buffer extension and mutilply rendering target. Live Demo github

CUDA GPU Path Tracing

A Monte Carlo Path Tracing developed on CUDA. Techniques used inlcude Supersampling, Frsenel Refraction, OBJ loader, Depth of Field, Motion Blur, Texture Map, Interactive Camera and SubSurface Scattering. For more info about this project, please go to my github

GPU Ray Tracing

I implemented raycast from camera, phong lighting for one point light source, diffuse lambertian surfaces, raytraced shadows, cube intersection testing and sphere surface point sampling. I also implemented specular reflection, soft shadows and area light, super sampled antialiasing and refraction. For more info, please go to my github

Bubble Simulation(Maya Plugin)

Our bubble simulation includes the bubble bubble interaction, bubble liquid interaction and bubble solid interaction. We also made a tight integration in Maya using the Maya Fluid as the water part for bubble.

Fire Simulation(Group Project)

In fire simulation, we used level set method to track the boundary of fuel and ghost velocity to solve the discontinuity on the surface between hot gas product and the fuel.

Cloth Simulation

The cloth simulation project is using Position Based Dynamics method for simulation. The cloth simulation project is using Position Based Dynamics method for simulation. It includes stretch, blending and fixed point constraints.

Smoke Simulation

Smoke simulation is based on the SIGGRAPH 2007 course note.It's MAC grid system. I implemented the Modified Incomplete Cholesky for preconditioner to reduce the iteration count when solving the pressure equations.

Jello Simulation

Jello Cube is using Mass-Spring system and coded in C++. I implemented the structural spring, shear spring and bend spring. For the collision, I am using penalty force method combined with the impulse force in order to make the collision more genuine.
